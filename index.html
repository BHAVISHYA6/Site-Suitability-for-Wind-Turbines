Mathematical Basis
Haversine: d = 2R·atan2(√a, √(1−a))
Normalize: f = min(dist, 20km) / 20
f1 urban: (d_bldg + d_school + d_hosp) / 3
f2 water: (d_way + d_body) / 2
f3 wildlife: (d_wild + d_land) / 2
f4 power: min(d_lines, d_stations)
Softmax: P(c) = e^zc / Σe^zk
Loss: −Σ y·log(ŷ) (cross-entropy)
